# LearnYB 项目迭代规划

## 阶段总览

| 阶段 | 名称 | 目标 |
|------|------|------|
| S1 | ✅ 音标表核心 | 48个音标交互式卡片 + 对比练习 + 进度跟踪 — 已完成 |
| S2 | 听力辨音练习 | 听音选音标、看音标选单词的交互练习 |
| S3 | 跟读录音 | 录音对比、波形可视化、发音评分 |
| S4 | 学习路径 | 7阶段课程体系、技能树、每日任务 |
| S5 | 打卡与激励 | 连续打卡、XP系统、成就徽章 |
| S6 | 单词查询 | 输入单词显示音标拆分、链接到音标卡片 |

---

## S2: 听力辨音练习

### 功能
- 听音辨音：播放音标，从4个选项中选出正确音标
- 看音标读单词：显示音标，播放单词音频，用户选对应的单词
- 每题限时 10 秒，超时自动判错
- 每轮 10 题，结束后显示得分和错题回顾
- 支持按音标类型筛选练习（如只练双元音）
- Web Speech API 发音 + 语速调节

### 技术点
- 练习数据驱动（题目数组、选项生成算法）
- 倒计时计时器
- 结果统计和错题展示
- 路由切换

---

## S3: 跟读录音

### 功能
- 点击录制按钮开始录音（MediaRecorder API）
- 实时波形可视化（录制时显示）
- 录制完成后自动播放对比（先标准 → 再用户录音）
- 简单的匹配度判断（例词文本对比，Web Speech API 识别用户发音）
- 录音历史记录

### 技术点
- MediaRecorder API
- AudioContext / AnalyserNode 实时波形
- Web Speech API 语音识别（speechRecognition）
- Canvas 绘制波形

---

## S4: 学习路径

### 功能
- 7 阶段技能树可视化
- 每阶段包含具体音标列表和 unlock 条件
- 阶段解锁：前一个阶段完成度 ≥ 70% 自动解锁下一阶段
- 每阶段有"开始学习"按钮 → 进入该阶段的专项练习
- 阶段进度条和当前等级标识

### 技术点
- 技能树数据结构
- Canvas 或 SVG 绘制技能树
- 阶段状态管理（locked/unlocked/in-progress/completed）
- 进度计算逻辑

---

## S5: 打卡与激励

### 功能
- 每日打卡（日历视图，标记已打卡日期）
- 连续打卡天数（streak fire）
- XP 系统：完成练习获得 XP，累计升级
- 成就徽章：首次打卡、连续 7 天、完成所有元音等
- 学习统计：总练习次数、总用时、各音标掌握度
- 全部 localStorage 持久化

### 技术点
- 打卡日历 UI
- Streak 计算（考虑时区、断签规则）
- XP 和等级计算
- 成就系统数据结构
- 统计图表（可用纯 CSS 实现简单柱状图）

---

## S6: 单词查询

### 功能
- 输入框输入英文单词
- 显示音标拆分（逐音标注）
- 每个音标注解链接 → 跳转到该音标详情
- 显示单词发音（Web Speech API）
- 支持英式/美式切换提示
- 搜索历史记录

### 技术点
- 简单音标拆分算法（字符串匹配）
- 搜索输入防抖
- 历史记录 localStorage

---

## 开发原则

- 每阶段可独立运行、独立提交
- 纯前端，零后端依赖
- 数据驱动 UI，减少硬编码
- 移动端优先响应式
